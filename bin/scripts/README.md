# python3 scripts



## FindLargest.py

###Find all files in path, display number of the largest files.
####Usage:  
```./FindLargest.py [path] [number-of-files-to-display] 
Optional(--psw=[paths,to,exclude]```

####Example: 
```Example: ./FindLargest.py /home 10 --psw=/dev,/sys,/proc 
--ere='^.*\.txt$'```
